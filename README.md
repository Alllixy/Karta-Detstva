Карта детства

О проекте
Проект представляет собой веб-приложение для поиска кружков и секций дополнительного образования. Родители могут найти занятие для ребенка по возрасту, цене, району и виду активности.

Функции
фильтрация кружков по городу, району, возрасту, типу активности и цене;
карта с отображением кружков и построением маршрута;
админ-панель для добавления, редактирования и удаления кружков;
адаптация под смартфон, планшет, Smart TV и smart-часы.
